あなたは法人税法の教材分析の専門家です。

以下の計算テキストPDFを分析し、2つの出力を生成してください。

## 出力1: 構造分析（Markdown）

教材の全体構造を以下の形式で出力してください:

```markdown
# {教材名} 構造分析

## 章立て
- 第1章: {章タイトル}（p.XX-XX）
  - 第1節: ...
  - 第2節: ...

## 学習構造（テキスト向け）
- 導入概念: {この章で前提となる考え方}
- 計算パターン: {章で扱う代表的な計算型}
- 章末整理: {まとめ・チェックポイントの有無}

## 論点カバレッジ
- 損金算入系: {該当する論点を列挙}
- 益金不算入系: ...
- 税額計算系: ...

## 特記事項
{教材の特徴、重点分野、計算手順の説明スタイルなど}
```

## 出力2: 論点リスト（JSON）

以下のJSON形式で、全章・全節を論点に分類してください。
JSONは必ず ```json ``` で囲んでください。

```json
{
  "source_name": "{出版社} {教材名}",
  "source_type": "計算テキスト",
  "publisher": "{出版社名}",
  "total_topics": {抽出した論点数},
  "topics": [
    {
      "topic_id": "{論点ID: サブカテゴリ_詳細形式。例: 減価償却_普通}",
      "name": "{論点の日本語名}",
      "category": "{大分類: 所得計算/損金算入/益金不算入/税額計算/グループ法人/組織再編/国際課税/その他}",
      "subcategory": "{中分類: 減価償却/引当金/受取配当 等}",
      "type": ["計算", "テキスト"],
      "conditions": ["{関連条文: 法XX, 令XX 形式}"],
      "section_refs": ["{参照箇所: 第X章第Y節, p.XX-YY}"],
      "calc_steps_outline": ["{計算手順の骨子: 1.対象額確定 2.限度額計算 3.損金算入額判定}"],
      "page_range": "{開始ページ}-{終了ページ}",
      "keywords": ["{この論点の重要キーワード}"],
      "importance": "{A/B/C: 出題頻度・配点から判定。A=頻出, B=標準, C=稀}",
      "related": ["{関連する他の論点のtopic_id}"]
    }
  ]
}
```

### topic_id の命名規則
- `{サブカテゴリ}_{詳細}` 形式（例: 減価償却_普通, 減価償却_特別, 受取配当_関連法人）
- 同じ論点が複数箇所に出てもtopic_idは1つ（section_refsに全参照箇所を入れる）

### category の選択肢（必ずこの中から選ぶ）
所得計算 / 損金算入 / 益金不算入 / 税額計算 / グループ法人 / 組織再編 / 国際課税 / その他

### importance の判定基準
- A: 本試験で毎年出題 or 配点が大きい
- B: 2〜3年に1回程度出題
- C: 稀にしか出ない or 細かい論点
