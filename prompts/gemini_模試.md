あなたは法人税法の教材分析の専門家です。

以下の模試PDFを分析し、2つの出力を生成してください。

## 出力1: 構造分析（Markdown）

教材の全体構造を以下の形式で出力してください:

```markdown
# {教材名} 構造分析

## 試験構成
- 第1問: {大問タイトル}（p.XX-XX）
  - 設問1: ...
  - 設問2: ...

## 出題傾向分析
- 頻出論点: {過去の出題傾向と一致する領域}
- 新傾向論点: {近年増加している切り口}
- 時間配分難所: {得点効率が低下しやすい箇所}

## 論点カバレッジ
- 損金算入系: {該当する論点を列挙}
- 益金不算入系: ...
- 税額計算系: ...

## 特記事項
{模試としての難易度、理論と計算の比重、合否を分ける論点など}
```

## 出力2: 論点リスト（JSON）

以下のJSON形式で、全設問を論点に分類してください。
JSONは必ず ```json ``` で囲んでください。

```json
{
  "source_name": "{出版社} {教材名}",
  "source_type": "模試",
  "publisher": "{出版社名}",
  "total_problems": {総問題数},
  "total_score": {満点},
  "passing_score": {合格基準点},
  "topics": [
    {
      "topic_id": "{論点ID: サブカテゴリ_詳細形式。例: 減価償却_普通}",
      "name": "{論点の日本語名}",
      "category": "{大分類: 所得計算/損金算入/益金不算入/税額計算/グループ法人/組織再編/国際課税/その他}",
      "subcategory": "{中分類: 減価償却/引当金/受取配当 等}",
      "type": ["計算", "理論", "模試"],
      "conditions": ["{関連条文: 法XX, 令XX 形式}"],
      "problem_numbers": ["{問題番号のリスト}"],
      "page_range": "{開始ページ}-{終了ページ}",
      "keywords": ["{この論点の重要キーワード}"],
      "importance": "{A/B/C: 出題頻度・配点から判定。A=頻出, B=標準, C=稀}",
      "related": ["{関連する他の論点のtopic_id}"]
    }
  ]
}
```

### topic_id の命名規則
- `{サブカテゴリ}_{詳細}` 形式（例: 減価償却_普通, 減価償却_特別, 受取配当_関連法人）
- 同じ論点が複数箇所に出てもtopic_idは1つ（problem_numbersに全問題番号を入れる）

### category の選択肢（必ずこの中から選ぶ）
所得計算 / 損金算入 / 益金不算入 / 税額計算 / グループ法人 / 組織再編 / 国際課税 / その他

### importance の判定基準
- A: 本試験で毎年出題 or 配点が大きい
- B: 2〜3年に1回程度出題
- C: 稀にしか出ない or 細かい論点
