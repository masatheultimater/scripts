あなたは法人税法の教材分析の専門家です。

以下の理論テキストPDFを分析し、2つの出力を生成してください。

## 出力1: 構造分析（Markdown）

教材の全体構造を以下の形式で出力してください:

```markdown
# {教材名} 構造分析

## 理論体系マップ
- 第1編: {体系名}
  - 第1章: {章タイトル}（p.XX-XX）
    - 第1節: ...
    - 第2節: ...

## 柱書き（理論の中核）
- {柱1: 例外を含む基本原則}
- {柱2: 判定要件のフレーム}
- {柱3: 他制度との接続}

## 論点カバレッジ
- 損金算入系: {該当する論点を列挙}
- 益金不算入系: ...
- 税額計算系: ...

## 特記事項
{理論の流れ、頻出論証、誤解しやすい対比ポイントなど}
```

## 出力2: 論点リスト（JSON）

以下のJSON形式で、全章・全節を論点に分類してください。
JSONは必ず ```json ``` で囲んでください。

```json
{
  "source_name": "{出版社} {教材名}",
  "source_type": "理論テキスト",
  "publisher": "{出版社名}",
  "total_topics": {抽出した論点数},
  "topics": [
    {
      "topic_id": "{論点ID: サブカテゴリ_詳細形式。例: 受取配当_益金不算入要件}",
      "name": "{論点の日本語名}",
      "category": "{大分類: 所得計算/損金算入/益金不算入/税額計算/グループ法人/組織再編/国際課税/その他}",
      "subcategory": "{中分類: 減価償却/引当金/受取配当 等}",
      "type": ["理論"],
      "conditions": ["{関連条文: 法XX, 令XX 形式}"],
      "pillars": ["{柱書き: この論点の根幹となる理論要素}"],
      "keywords": [
        "{上位概念キーワード}",
        "{要件キーワード}",
        "{例外キーワード}",
        "{比較・対立概念キーワード}"
      ],
      "memorization_points": ["{暗記ポイント: 要件の語順、数値基準、例外条件など}"],
      "page_range": "{開始ページ}-{終了ページ}",
      "importance": "{A/B/C: 出題頻度・配点から判定。A=頻出, B=標準, C=稀}",
      "related": ["{関連する他の論点のtopic_id}"]
    }
  ]
}
```

### topic_id の命名規則
- `{サブカテゴリ}_{詳細}` 形式（例: 減価償却_普通, 減価償却_特別, 受取配当_関連法人）
- 同じ論点が複数箇所に出てもtopic_idは1つ（重複は統合して整理）

### category の選択肢（必ずこの中から選ぶ）
所得計算 / 損金算入 / 益金不算入 / 税額計算 / グループ法人 / 組織再編 / 国際課税 / その他

### importance の判定基準
- A: 本試験で毎年出題 or 配点が大きい
- B: 2〜3年に1回程度出題
- C: 稀にしか出ない or 細かい論点
