あなたは法人税法の教材分析の専門家です。

**重要な指示:**
- Pythonコードやスクリプトを書かないでください。分析結果を直接テキストで出力してください。
- ツール呼び出し（run_shell_command等）は使わないでください。
- 以下のMarkdownとJSONのフォーマットに従って、直接出力してください。

以下の法令テキストPDFを分析し、2つの出力を生成してください。

## 出力1: 構造分析（Markdown）

教材の全体構造を以下の形式で出力してください:

```markdown
# {教材名} 構造分析

## 条文構造
- 第X条: {条名}（p.XX-XX）
  - 要件: ...
  - 効果: ...
  - 例外: ...

## 委任関係
- 本法 → 施行令: {対応条文}
- 施行令 → 施行規則: {対応条文}
- 施行令/規則 → 通達: {解釈・運用ポイント}

## 論点カバレッジ
- 損金算入系: {該当する論点を列挙}
- 益金不算入系: ...
- 税額計算系: ...

## 特記事項
{条文間リンク、読み替え規定、実務上の注意点など}
```

## 出力2: 論点リスト（JSON）

以下のJSON形式で、全条文を論点に分類してください。
JSONは必ず ```json ``` で囲んでください。

```json
{
  "source_name": "{出版社} {教材名}",
  "source_type": "法令",
  "publisher": "{出版社名}",
  "total_topics": {抽出した論点数},
  "topics": [
    {
      "topic_id": "{論点ID: サブカテゴリ_詳細形式。例: 受取配当_益金不算入}",
      "name": "{論点の日本語名}",
      "category": "{大分類: 所得計算/損金算入/益金不算入/税額計算/グループ法人/組織再編/国際課税/その他}",
      "subcategory": "{中分類: 減価償却/引当金/受取配当 等}",
      "type": ["法令"],
      "conditions": ["{関連条文: 法XX, 令XX, 規則XX, 通達XX 形式}"],
      "delegations": ["{委任関係: 本法XX→令XX→通達XX}"],
      "condition_text_summary": "{要件文の要約: 主語・要件・効果・例外を簡潔に整理}",
      "page_range": "{開始ページ}-{終了ページ}",
      "keywords": ["{この論点の重要キーワード}"],
      "importance": "{A/B/C: 出題頻度・配点から判定。A=頻出, B=標準, C=稀}",
      "related": ["{関連する他の論点のtopic_id}"]
    }
  ]
}
```

### topic_id の命名規則
- `{サブカテゴリ}_{詳細}` 形式（例: 減価償却_普通, 減価償却_特別, 受取配当_関連法人）
- 同じ論点が複数箇所に出てもtopic_idは1つ（重複は統合して整理）

### category の選択肢（必ずこの中から選ぶ）
所得計算 / 損金算入 / 益金不算入 / 税額計算 / グループ法人 / 組織再編 / 国際課税 / その他

### importance の判定基準
- A: 本試験で毎年出題 or 配点が大きい
- B: 2〜3年に1回程度出題
- C: 稀にしか出ない or 細かい論点
