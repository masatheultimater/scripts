あなたは法人税法の教材分析の専門家です。

**重要な指示:**
- Pythonコードやスクリプトを書かないでください。分析結果を直接テキストで出力してください。
- ツール呼び出し（run_shell_command等）は使わないでください。
- 以下のMarkdownとJSONのフォーマットに従って、直接出力してください。

以下の実務書PDFを分析し、2つの出力を生成してください。

## 出力1: 構造分析（Markdown）

教材の全体構造を以下の形式で出力してください:

```markdown
# {書名} 構造分析

## 全体構成
- 第X編/章: {タイトル}（p.XX-XX）
  - 第X節: ...
  - 第X節: ...

## 主要テーマ
- {この書籍が重点的に解説しているテーマ}
- {実務上の論点・判断ポイント}

## 論点カバレッジ
- 損金算入系: {該当する論点を列挙}
- 益金不算入系: ...
- 税額計算系: ...

## 特記事項
{実務書としての特徴、通達解釈、判例・裁決例の有無、計算例の充実度など}
```

## 出力2: 論点リスト（JSON）

以下のJSON形式で、書籍内の全論点を分類してください。
JSONは必ず ```json ``` で囲んでください。

```json
{
  "source_name": "{出版社} {書名}",
  "source_type": "実務書",
  "publisher": "{出版社名}",
  "total_topics": {抽出した論点数},
  "topics": [
    {
      "topic_id": "{論点ID: サブカテゴリ_詳細形式。例: 交際費_5000円基準}",
      "name": "{論点の日本語名}",
      "category": "{大分類: 所得計算/損金算入/益金不算入/税額計算/グループ法人/組織再編/国際課税/その他}",
      "subcategory": "{中分類: 減価償却/引当金/受取配当 等}",
      "type": ["実務"],
      "conditions": ["{関連条文: 法XX, 令XX, 通達XX 形式}"],
      "practical_points": ["{実務上の判断ポイント・注意事項}"],
      "page_range": "{開始ページ}-{終了ページ}",
      "keywords": ["{この論点の重要キーワード}"],
      "importance": "{A/B/C: 試験出題頻度から判定。A=頻出, B=標準, C=稀}",
      "related": ["{関連する他の論点のtopic_id}"]
    }
  ]
}
```

### topic_id の命名規則
- `{サブカテゴリ}_{詳細}` 形式（例: 減価償却_普通, 減価償却_特別, 受取配当_関連法人）
- 同じ論点が複数箇所に出てもtopic_idは1つ（page_rangeに代表的な参照箇所を入れる）
- 既存の大原テキスト・問題集と同じtopic_idを使い、新規論点のみ新IDを付与

### category の選択肢（必ずこの中から選ぶ）
所得計算 / 損金算入 / 益金不算入 / 税額計算 / グループ法人 / 組織再編 / 国際課税 / その他

### importance の判定基準
- A: 本試験で毎年出題 or 配点が大きい
- B: 2〜3年に1回程度出題
- C: 稀にしか出ない or 細かい論点
